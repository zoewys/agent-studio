二、视觉层级规则

按信息重要性从高到低排列。AI 正式回复是视觉中心，工具调用仅作为辅助信息低调展示。

层级元素样式目的
1 · 最突出 AI 正式消息 白色卡片 + 左侧铜青色条 + 阴影 用户阅读的核心内容
2 · 次级 用户消息 右对齐 + 半透明铜青底 对话上下文
3 · 辅助 权限请求 黄色边框独立卡片 需要用户操作时醒目
4 · 最弱 工具调用 / 文件变更 单行内联，无背景，仅细色条 执行痕迹，扫一眼即可
5 · 隐藏 思考中 单行可折叠，默认收起 需要时才展开

实现要点：动画使用 CSS 完成，不依赖 JS。仅在状态变化时触发动画，避免过度炫技。 

// React 中触发动画：给新消息添加动画类，动画结束后移除
const MessageCard = ({ event, isNew }) => (
  <div className={
    isNew ? 'msg-ai msg-ai-animate' : 'msg-ai'
  }>
    ...
  </div>
);

// CSS 中定义动画
.msg-ai-animate {
  animation: msgAppear 0.3s cubic-bezier(0.16, 1, 0.3, 1) forwards;
}
